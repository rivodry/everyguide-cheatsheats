# JavaScript Cheatsheet

## Variables
- **Const:** `const name = "value"`
- **Let:** `let counter = 0`
- **Var:** `var legacy = "old"`

## Functions
```javascript
const greet = (name) => {
  return `Hello, ${name}!`;
};
```

## Arrays
- `map()` - Transform each element
- `filter()` - Filter elements
- `reduce()` - Reduce to single value

## Objects
```javascript
const person = {
  name: "John",
  age: 30,
  greet() {
    console.log("Hi!");
  }
};
```
